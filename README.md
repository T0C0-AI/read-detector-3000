# 읽씹 판독기 3000 🔮

> 그 카톡, 답장해? 말아? — 병맛 점쟁이 컨셉의 **AI 카톡 답장 판독기**

메시지나 대화를 넣으면 AI가 **"답장 ㄱㄱ / 읽씹 ㄱㄴ"** 를 약올리는 톤으로 판독해주고,
썸 상대와의 관계까지 분석해주는 장난감 웹앱입니다. (과학적 근거는 1도 없습니다 😎)

---

## ✨ 기능

- 🔍 **사전 심문** — 사이·만난 기간·목표·연락 스타일 등 맥락을 칩으로 골라 정확도 ↑
- 🎰 **무료 즉석 판독** — 키워드 기반, 서버·키 없이 즉시 (비용 0)
- 💘 **AI 썸 판독** — 호감도 %·등급(SSR~N)·관계 진단
- 💬 **답장 3종 추천** — 쿨하게 / 귀엽게 / 밀당 버전
- ✍️ **답장 첨삭** — 내가 쓴 답장 점수 매기고 더 나은 버전 제안
- 🔮 **미래 예측** — 이 답장 보내면 상대가 어떻게 반응할지
- 📸 **카톡 캡처 판독** — 스크린샷을 올리면 AI가 읽어서 분석 (비전)
- 📤 **결과 공유 카드** · 📁 **수집 도감** · 📈 **호감도 추적**
- 🎆 **SSR 폭죽 연출** — 높은 등급이 뜨면 색종이가 터집니다

## 🚀 실행 방법

### 1. 준비물
- [Node.js](https://nodejs.org) **20.12 이상** (외부 패키지 설치 불필요 — 의존성 0개)
- [Anthropic(Claude) API 키](https://console.anthropic.com) — AI 판독 기능에만 필요

### 2. 키 설정
```bash
cp .env.example .env
# .env 파일을 열어 ANTHROPIC_API_KEY 에 본인 키를 붙여넣기
```

### 3. 실행
```bash
npm start
# 브라우저에서 http://localhost:8787 접속
```

> 🎰 **무료 즉석 판독**은 키 없이도 동작합니다. AI 기능(💘 썸 판독·✍️ 첨삭·🔮 예측·📸 캡처)만 키가 필요해요.

## 🛠 기술 스택

- **백엔드**: Node.js 내장 `http` + `fetch` (외부 의존성 0개)
- **AI**: Anthropic Claude (Haiku) — 빠르고 저렴한 모델
- **프론트**: Vanilla HTML / CSS / JS (빌드 도구 없음)

파일이라곤 `index.html`(화면) + `server.mjs`(서버) 둘이 전부입니다.

---

## 🔒 보안

- API 키는 **`.env`에만** 저장되고 `.gitignore`로 커밋에서 제외됩니다 (브라우저에 절대 노출 안 됨)
- 입력 길이 제한 + 간이 Rate Limit(요금 폭탄 방지) + 호출 실패 시 재시도 1회
- AI 응답은 화면에 **텍스트로만** 출력해 XSS 차단
- 업로드 이미지는 형식(png·jpg·webp·gif)·용량(5MB) 검증
